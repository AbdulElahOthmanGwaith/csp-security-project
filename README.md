# 🛡️ مشروع سياسة أمان المحتوى (CSP) - الإصدار المطور 2026+ 🚀

## 🌟 نظرة عامة | Overview
هذا المشروع هو المرجع العربي الأول والشامل لتطبيق **سياسة أمان المحتوى (Content Security Policy)**. يهدف إلى تمكين المطورين من حماية تطبيقات الويب الخاصة بهم ضد هجمات XSS و Clickjacking وغيرها من الثغرات الأمنية باستخدام أحدث المعايير التقنية لعام 2026.

## 🚀 الميزات الجديدة | New Features
*   **CSP Level 3 & 4 Support**: دعم كامل لأحدث توجيهات الأمان وتحديثات عام 2026.
*   **Interactive CSP Generator**: أداة تفاعلية جديدة لتوليد سياسات أمان مخصصة بسهولة.
*   **Security Policy Enforcement**: إضافة طبقة حماية ذاتية للمشروع باستخدام Trusted Types.
*   **Trusted Types**: حماية متقدمة ضد ثغرات DOM-based XSS.
*   **Modern UI & Dark Mode**: واجهة مستخدم عصرية متجاوبة بالكامل مع دعم الوضع الليلي.
*   **Comprehensive Examples**: أكثر من 50 مثالاً عملياً يغطي حالات الاستخدام المختلفة.

## 🛠️ الأدوات المضمنة | Included Tools
1.  **CSP Builder (Generator)**: منشئ سياسات تفاعلي سهل الاستخدام (جديد!).
2.  **Security Scanner**: فحص سريع لرؤوس الأمان (Security Headers).
3.  **Code Analyzer**: تحليل الكود المصدري لاكتشاف الممارسات غير الآمنة.

## 📂 هيكل المشروع | Project Structure
*   `/js`: يحتوي على المتحكمات (Controllers) للميزات التفاعلية ومولد السياسات.
*   `/styles`: ملفات التنسيق الحديثة والمتجاوبة.
*   `/imgs`: الرسوم التوضيحية والخرائط الذهنية للأمان.

## 📚 كيف تبدأ | How to Start
ببساطة قم بفتح ملف `index.html` في متصفحك، أو استخدم خادم محلي:
```shell

python3 -m http.server 8000
```
